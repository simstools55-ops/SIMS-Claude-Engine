# Claude Engine Quality Evaluation v1.0

## Evaluation Items

### SEO Accuracy
- Main query matches search intent.
- Sub queries support the main query.
- Title tag is click-oriented and not misleading.
- Meta description is concise and useful.

### Beginner Usability
- Explains what to change.
- Explains why to change it.
- Explains where to paste it in WordPress.
- Provides a checklist.

### Exchange Compliance
- Uses AI Exchange v1.0 structure.
- Does not omit required fields.
- Keeps the result easy to paste back into SIMS.
