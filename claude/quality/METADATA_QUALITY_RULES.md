# Metadata Quality Rules

## H1

- Clear topic.
- Matches article content.
- Usually includes Main Query or close variant.

## Title tag

- Written for search result clicks.
- Should include Main Query naturally.
- Should not be identical to H1 unless appropriate.

## Meta description

- Explains what the reader gets.
- Encourages click without exaggeration.
- Should include Main Query or related wording.

## Smartphone description

- First 50 to 60 Japanese characters should communicate the core value.
- Avoid long introductions.
