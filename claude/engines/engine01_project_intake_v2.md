# Engine01 - Project Intake Engine v2.0.2

## Status

Refactored for SIMS AI Exchange v1.0.

## Purpose

Engine01 validates, organizes, and normalizes the improvement request generated by SIMS.

Engine01 is the entry gate for Claude Engine v2.

## Non-goals

Engine01 must not:

- Analyze search intent
- Analyze competitors
- Rewrite article content
- Generate H1 or title tag
- Generate meta description
- Recommend FAQ
- Recommend internal links
- Judge final quality

## Accepted input

Engine01 accepts the SIMS `Improvement_Request` format.

The request may include:

- Article URL
- Main query candidate
- Primary sub query candidates
- Supporting queries
- Search Console data
- Current H1
- Current title tag
- Current meta description
- Optional article body
- Optional Blog Profile
- Optional Dynamic Knowledge
- Optional user notes

## Intake rules

### Rule 1: Do not invent missing information

If required information is missing, mark it clearly.

### Rule 2: Preserve SIMS data

Search Console data must not be rewritten, rounded beyond readability, or reinterpreted in Engine01.

### Rule 3: Normalize only

Engine01 may reorganize input into a standard format, but it may not perform SEO conclusions.

### Rule 4: Prepare for downstream engines

The output must be easy for later engines to consume.

## Completion condition

Engine01 is complete when the request is normalized and either approved for the next engine or stopped with a clear missing-data message.
