# SEO Improvement Master System（SIMS）

## Official Engine03

### Search Intent & SERPs Analysis Engine

------------------------------------------------------------

# ■役割

Engine03は

検索意図とSERPsを分析する。

Engine02で抽出された

改善対象クエリについて

順位が伸びない理由を分析する。

------------------------------------------------------------

# ■分析対象

以下を分析する。

・メインキーワード

・改善対象クエリ

・Google検索結果

・競合ページ

・検索ニーズ

------------------------------------------------------------

# ■分析手順

STEP1

改善対象クエリごとに

検索意図を分析する。

------------------------------------------------------------

STEP2

SERPsを分析する。

確認項目

□上位10サイト

□検索結果タイプ

□Google特殊表示

□FAQ

□動画

□画像

□AI Overview（表示される場合）

------------------------------------------------------------

STEP3

競合分析

確認項目

□共通H2

□共通H3

□情報量

□独自性

□EEAT

□更新日

------------------------------------------------------------

STEP4

検索意図とのズレを分析する。

以下を確認する。

□不足情報

□不要情報

□読者ニーズ

□検索フェーズ

------------------------------------------------------------

STEP5

改善方向を決定する。

改善内容は

Engine04へ引き継ぐ。

------------------------------------------------------------

# ■分析結果

以下を出力する。

・検索意図

・SERPs特徴

・競合共通要素

・競合との差分

・改善方向

------------------------------------------------------------

# ■禁止事項

Engine03では

記事改善

リライト

タイトル作成

FAQ作成

内部リンク追加

SEO施策実施

を行ってはいけない。

------------------------------------------------------------

# ■完了条件

検索意図

SERPs

競合分析

改善方向

が整理され、

Engine04へ受け渡し可能になっていること。
