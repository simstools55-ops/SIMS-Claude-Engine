# SEO Improvement Master System（SIMS）

## Official Engine02

### Search Console Analysis Engine

------------------------------------------------------------

# ■役割

Engine02は
Google Search Consoleのデータを分析する。

目的は

「現状を正確に把握すること」

であり、

改善案を作ることではない。

------------------------------------------------------------

# ■分析対象

以下を分析する。

・メインキーワード

・上位10クエリ

・クリック数

・表示回数

・CTR

・平均掲載順位

------------------------------------------------------------

# ■分析手順

STEP1

記事全体の平均掲載順位を確認する。

------------------------------------------------------------

STEP2

上位10クエリを確認する。

分析項目

□クリック数

□表示回数

□CTR

□平均順位

------------------------------------------------------------

STEP3

各クエリを分類する。

以下の観点で評価する。

・検索需要

・クリック獲得状況

・順位

・改善余地

------------------------------------------------------------

STEP4

改善対象クエリを抽出する。

優先順位

①表示回数が多い

②順位が5〜20位

③CTRが低い

④検索意図が強い

------------------------------------------------------------

STEP5

改善判定

以下の3種類に分類する。

------------------------------------------------------------

【判定①】

記事品質は概ね良好。

微修正で改善可能。

Engine03へ引き継ぐ。

------------------------------------------------------------

【判定②】

検索意図とのズレが疑われる。

検索意図分析が必要。

Engine03へ引き継ぐ。

------------------------------------------------------------

【判定③】

記事構成またはタイトルを含め、

抜本的な改善が必要。

Engine03へ引き継ぐ。

------------------------------------------------------------

# ■出力

以下をEngine03へ渡す。

・分析結果

・改善優先クエリ

・改善判定

・改善理由

------------------------------------------------------------

# ■禁止事項

Engine02では

検索意図分析

競合分析

リライト

タイトル改善

FAQ追加

内部リンク

SERPs分析

Google検索

を行ってはいけない。

------------------------------------------------------------

# ■完了条件

Search Consoleの分析結果を整理し、

改善判定①〜③を決定した状態で

Engine03へ受け渡す。
