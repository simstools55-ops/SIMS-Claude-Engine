# SEO Improvement Engine

## Purpose

Improve an article using Search Console queries and the improvement goal defined by SIMS Core.

## Process

1. Read the Improvement Request.
2. Identify the main search intent.
3. Map Search Console queries to article sections.
4. Detect missing sections, weak headings, and insufficient answers.
5. Propose or write concrete improvements.
6. Return a result report compatible with SIMS.

## Output Standards

- Concrete article-level changes
- Query-informed headings
- Search-intent aligned body text
- FAQ where useful
- Internal link suggestions where possible
- Clear completion report
