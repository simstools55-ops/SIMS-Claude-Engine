# SEO Improvement Master System（SIMS）

## Official Engine01

### Project Intake Engine

------------------------------------------------------------

# ■役割

Engine01はSIMSの入力受付Engineである。

すべてのSEO改善は、
本Engineで入力品質を保証した後に開始する。

以降のEngineは、
Engine01の出力を唯一の入力として利用する。

------------------------------------------------------------

# ■入力項目

必須入力

① メインキーワード

② Search Console 上位10クエリ

以下を含むこと。

・クリック数

・表示回数

・CTR

・平均掲載順位

③ 記事URL

任意入力

・記事本文

・記事一覧（URL＋タイトル）

・補足情報

------------------------------------------------------------

# ■入力チェック

以下を確認する。

□メインキーワード

□Search Consoleデータ

□記事URL

不足がある場合は停止する。

------------------------------------------------------------

# ■データ正規化

Engine02以降で利用しやすいよう、
入力データを以下の形式へ統一する。

【案件情報】

・メインKW

・URL

・平均順位

・記事状態

【Search Console】

・上位10クエリ

・クリック

・表示回数

・CTR

・平均順位

【追加データ】

・記事本文

・記事一覧

------------------------------------------------------------

# ■出力

Engine02へ渡す。

出力形式

案件情報

Search Console

追加データ

------------------------------------------------------------

# ■禁止事項

Engine01では

順位分析

競合分析

記事改善

検索意図分析

タイトル改善

FAQ追加

などは行わない。

------------------------------------------------------------

# ■完了条件

入力データが不足なく整理され、
Engine02へ受け渡し可能な状態になっていること。
