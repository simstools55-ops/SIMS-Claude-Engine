# Result Quality Checklist

Before returning an Improvement Result, confirm:

- [ ] Main Query is identified.
- [ ] Sub Queries are identified.
- [ ] H1 is provided.
- [ ] Title Tag is provided.
- [ ] Meta Description is provided.
- [ ] Smartphone Description is provided.
- [ ] Article changes are specific.
- [ ] Beginner explanation is included.
- [ ] WordPress application steps are included.
- [ ] Completion checklist is included.
- [ ] Reasoning Summary is concise and user-facing.
