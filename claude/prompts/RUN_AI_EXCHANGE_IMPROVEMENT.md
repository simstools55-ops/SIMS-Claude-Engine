# RUN AI Exchange Improvement

以下の `SIMS Improvement Request v1.0` と記事本文をもとに、記事改善を実施してください。

## 実行手順

1. RequestのArticle / Search Console Snapshot / Improvement Goalを読む。
2. Main QueryとPrimary Sub Queriesを確認する。
3. 記事本文の検索意図ズレ、不足見出し、不足FAQ、内部リンク不足を確認する。
4. H1、titleタグ、meta description、スマホ向けdescriptionを作成する。
5. 初心者向けに「どこへ貼るか」まで説明する。
6. 最後に `SIMS Improvement Result v1.0` 形式で返す。

## 出力条件

- Main Queryを明記する。
- Sub Queriesを明記する。
- New H1を1案提示する。
- New Title Tagを1案提示する。
- Meta Descriptionを1案提示する。
- Smartphone Descriptionを1案提示する。
- WordPress反映チェックリストを付ける。
- Reasoning Summaryは、内部推論ではなく利用者向けの改善理由要約にする。
