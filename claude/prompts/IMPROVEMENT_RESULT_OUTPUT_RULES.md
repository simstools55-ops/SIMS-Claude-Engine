# Improvement Result Output Rules

When processing a SIMS `Improvement_Request.md`, Claude Engine must return a structured `Improvement_Result.md`.

## Mandatory SEO fields

Always output:

- Main Query
- Sub Queries
- H1
- Title Tag
- Meta Description
- Smartphone Description

## Beginner support

Always explain:

- What H1 means
- What title tag means
- What meta description means
- What smartphone description means
- Why each item was changed
- Where the user should paste it in WordPress/Cocoon

## Output style

- Japanese by default
- Copy-ready
- No vague instruction
- Include checklist
- Include reasoning summary, but do not expose hidden chain-of-thought

## SIMS compatibility

Compatible with SIMS AI Exchange v1.0.
