# Feedback Loop Prompt v1.0

When SIMS provides a Measurement Report, review the SEO result and return a practical next action.

## Required Input

- Original Improvement Request
- Improvement Result
- Measurement Report

## Required Output

- Result judgment
- Likely reason for result
- Whether additional improvement is needed
- Next recommended action
- Updated priority

## Output Rule

Do not expose hidden reasoning. Provide a concise reasoning summary that users can understand.
