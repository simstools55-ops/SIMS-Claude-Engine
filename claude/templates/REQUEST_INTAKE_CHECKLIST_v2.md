# Request Intake Checklist v2.0

Before improving an article, verify:

- [ ] URL is present
- [ ] Main Query candidate is present
- [ ] Query table is present or user has explained why unavailable
- [ ] Current article body is present or user will paste it
- [ ] H1 / Title / Description are present or marked unknown
- [ ] Improvement goal is clear
- [ ] Blog context is present or marked minimal

If article body is missing, ask the user to paste it before full rewrite.
