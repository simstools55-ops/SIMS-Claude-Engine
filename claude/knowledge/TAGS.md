# TAG SYSTEM

このドキュメントは

SIMSで使用する改善タグの公式定義です。

改善タグは

・Engine9

・Engine10

・Dashboard

すべて共通で使用します。

タグ名は変更してはいけません。

追加する場合はMinor Versionを更新してください。

------------------------------------------------------------

# タグ利用ルール

タグは

実際に実施した改善だけ付与してください。

予定や提案だけでは付与しません。

タグは必要最小限とします。

同じ意味のタグを重複して付与してはいけません。
