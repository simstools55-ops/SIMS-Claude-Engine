# MVP Request Processing Test

## Test Input
Paste one Improvement_Request.md generated by SIMS.

## Expected Claude Output
Claude must return:

- Main Query
- Sub Queries
- H1 recommendation
- Title tag recommendation
- Meta description recommendation
- Smartphone description recommendation
- Article improvement summary
- Beginner explanation
- WordPress reflection checklist
- Improvement_Result.md compatible completion report

## Fail Conditions
- Output lacks SEO metadata.
- Output does not explain where to use H1, title tag, or description.
- Output cannot be copied back into SIMS.
