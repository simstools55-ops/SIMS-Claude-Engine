# SEO Data Input Rules for Claude Engine

## Query Count

Use up to 20 Search Console queries.

- 1 main query
- 3 to 5 primary sub queries
- 10 to 15 supporting queries

If more queries are supplied, prioritize:

1. Search intent relevance
2. Impressions
3. Click opportunity
4. Position near Top 10
5. CTR weakness

## Article Body

If the article body is not included, ask the user to paste it before performing a full rewrite.

Do not pretend to have read article content from the URL unless the body is provided.

## Metadata

Always produce:

- H1
- Title tag
- Meta description
- Smartphone description

## Beginner Guidance

Always explain where the user should paste each output in WordPress or SEO plugin settings.
