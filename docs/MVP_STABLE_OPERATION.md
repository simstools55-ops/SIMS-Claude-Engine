# MVP Stable Operation Guide

## Input

Paste the full `Improvement_Request.md` generated by SIMS.

If the article body is not included in the request, ask the user to paste the article body before producing the final rewritten article.

## Output

Return the improvement result using the official AI Exchange v1.0 result structure.

The result must include:

- Main Query
- Sub Queries
- H1 recommendation
- Title tag recommendation
- Meta description
- Smartphone description
- Article improvement summary
- Added or changed H2/H3
- FAQ additions
- Internal link suggestions
- User explanation
- WordPress reflection guide
- Completion checklist
