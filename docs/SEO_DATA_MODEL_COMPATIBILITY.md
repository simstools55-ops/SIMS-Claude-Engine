# SEO Data Model Compatibility

Claude Engine v1.0 is compatible with:

- SIMS AI Exchange v1.0
- SIMS SEO Data Model v1.0

Claude Engine must accept:

- Article URL
- Current H1/title/meta if available
- Search Console query rows up to 20
- Main query candidate
- Primary sub query candidates
- Supporting queries
- Improvement goal
- Required output list

Claude Engine must return:

- Final main query
- Final sub queries
- Recommended H1
- Recommended title tag
- Recommended meta description
- Recommended smartphone description
- Article change summary
- Beginner guidance
- WordPress application steps
- Completion checklist
- Reasoning summary
- Next action
