# Setup

## Claude Project Setup

1. Create a Claude Project.
2. Copy `claude/instructions/CLAUDE_PROJECT_INSTRUCTIONS.md` into Project Instructions.
3. Add files in `claude/engines/` to Project Knowledge.
4. Add files in `claude/knowledge/` to Project Knowledge.
5. Keep `claude/exchange-templates/` as reference templates.

## Recommended Repository Name

`SIMS-Claude-Engine`
