# Claude Project Setup

## 1. Project作成

Claudeで新規Projectを作成し、名前を `SIMS Claude Engine` にします。

## 2. Project Instructions

`claude/project-instructions/CLAUDE_PROJECT_INSTRUCTIONS_PROTOTYPE_BASELINE.md` の内容をProject Instructionsへ貼り付けます。

## 3. Knowledge追加

以下をProject Knowledgeへ追加します。

- `claude/engines/`
- `claude/knowledge/`
- `claude/prompts/`
- `claude/templates/`

## 4. 初回テスト

SIMSの `AI_Exchange_Requests` シートに生成された `RequestMarkdown` をClaudeへ貼り付け、対象記事本文も続けて貼ります。

Claudeが `SIMS Improvement Result v1.0` 形式で返すか確認してください。
