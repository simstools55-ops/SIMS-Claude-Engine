# Exchange Processor Specification v1.0

## 対応仕様

- SIMS AI Exchange Standard v1.0

## Claude Engineの責務

Claude Engineは、SIMSが作成した改善依頼を受け取り、記事改善を実行する。

## SIMSに返すデータ

- Main Query
- Sub Queries
- H1
- Title Tag
- Meta Description
- Smartphone Description
- FAQ
- 追加H2
- 追加H3
- 内部リンク
- 改善理由要約
- 次回改善提案

## 初心者対応

出力には必ず、用語説明、反映場所、チェックリストを含める。
