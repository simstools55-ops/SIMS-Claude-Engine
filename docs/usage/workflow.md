# Workflow

## Standard Flow

1. SIMS Core generates `Improvement_Request.md`.
2. User pastes the request into Claude Project.
3. Claude improves the article.
4. Claude returns article content and `Improvement_Result.md`.
5. User records the result in SIMS Core.
6. SIMS Core measures performance after the selected period.

## Important Principle

SIMS decides what should be improved. Claude decides how to improve the article.
