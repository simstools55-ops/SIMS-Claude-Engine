# Claude Engine v1.0 Production Operation

## Role

Claude Engine performs the article improvement work requested by SIMS.

## Input

Claude receives Improvement Request from SIMS.

The request may include:

- URL
- Main Query
- Sub Queries
- Search Console query table
- Current H1/title/description when available
- Article body pasted by the user
- Required outputs

## Output

Claude returns Improvement Result.

The result must include:

- Main Query
- Sub Queries
- Recommended H1
- Recommended Title Tag
- Recommended Meta Description
- Smartphone Description
- Article improvement summary
- Beginner explanation
- WordPress application steps
- Checklist
- Reasoning Summary

## Important rule

Claude must not only improve the article.
Claude must also explain what the user should copy and where to paste it.
