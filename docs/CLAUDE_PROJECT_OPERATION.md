# Claude Project Operation Guide

## Basic Flow

1. Open Claude Project for SIMS Claude Engine.
2. Paste `Improvement_Request.md` from SIMS.
3. Run the AI Exchange Execution Prompt.
4. Copy the generated `Improvement_Result.md`.
5. Paste or import the result into SIMS.
6. Apply the recommended changes in WordPress.

## Important Rule

Claude Engine improves the article. SIMS manages the improvement lifecycle.

Do not store Search Console credentials or private site credentials in Claude Project Knowledge.
