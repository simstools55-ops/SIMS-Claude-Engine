# Claude Engine AI Exchange MVP Usage

## 使い方

1. SIMSで生成した `SIMS Improvement Request v1.0` をClaude Projectへ貼り付けます。
2. 改善対象の記事本文を貼り付けます。
3. `RUN_AI_EXCHANGE_IMPROVEMENT.md` の手順で改善を実行します。
4. 最後に `SIMS Improvement Result v1.0` 形式で返します。
5. ユーザーはResultをSIMSへ貼り付けます。

## 対応仕様

- SIMS AI Exchange Standard v1.0
- SEO Data Model v1.0
- SIMS Improvement Request v1.0
- SIMS Improvement Result v1.0
