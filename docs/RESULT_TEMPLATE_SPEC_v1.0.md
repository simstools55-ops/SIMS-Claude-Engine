# Result Template Specification v1.0

This document defines Claude Engine output for SIMS AI Exchange v1.0.

## Required output

Claude must return `Improvement_Result.md` in Markdown.

## Human-readable design

The result must be readable by beginner bloggers. It must include:

- SEO output values
- Why each value was chosen
- How to reflect changes in WordPress
- Checklist
- Reasoning Summary

## Machine-readability

SIMS may parse key headings and fields. Therefore, section headings must remain stable.
