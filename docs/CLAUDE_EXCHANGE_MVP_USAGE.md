# Claude Exchange MVP Usage

## How to use

1. Paste the SIMS `Improvement_Request.md` into the Claude Project.
2. Ask Claude to process it with the SIMS Claude Exchange MVP Processor.
3. Claude returns `Improvement_Result.md`.
4. Copy the result back to SIMS or store it as the improvement record.

## Quality rule

The result must be understandable by a beginner and useful for an advanced SEO operator.
