# Claude Engine MVP UAT Guide

Use this guide to validate whether Claude Engine can process SIMS Improvement Requests.

1. Open Claude Project for SIMS-Claude-Engine.
2. Paste the Improvement_Request.md from SIMS.
3. Ask Claude to process it according to AI Exchange v1.0.
4. Confirm the returned result includes SEO metadata, explanation, and checklist.
5. Save the result as Improvement_Result.md.
